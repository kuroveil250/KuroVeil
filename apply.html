What we’re looking for

Consistent
Collab-ready
Clip-minded
Communicates
  • Consistency (stream/upload cadence you can actually maintain)
  • Good communication + teamwork
  • Willingness to collab and create clip-worthy moments
  • Fits a shadow/dark vibe (lore optional, not required)

Minimum expectations

  • At least 1 stream per week and 3 short-form uploads per week
  • Show up for scheduled collabs/events
  • Stay active in Discord and communicate clearly
  • No drama / no ghosting / no bait applications

If you can’t commit yet, that’s fine — apply later when you’re ready to move weekly.

What you get (real structure)

  • Weekly collab block (a fixed weekly slot so collabs actually happen)
  • Monthly event format (tournament / horror night / challenge series)
  • Clip review loop (hooks, pacing, titles, thumbnails)
  • Shared promo rules (retweet/repost habits + launch boosts)
  • Asset pack (basic overlays, panels, tags, posting templates)

How the process works

  • Submit the form
  • We review on a rolling basis
  • Short Discord chat (fit + goals)
  • If accepted: onboarding + first collab plan

Don’t worry about being “perfect.” We care about effort + consistency + being easy to work with.

FAQ

Do I need a model already? ›

No — but you must be actively working on your setup and able to stream consistently.

Can I join if I’m small? ›

Yes. Consistency + teamwork matter more than size. Founding Gen is about building momentum together.

Can I be pre-debut / PNG / faceless? ›

Yes, as long as you can start streaming consistently and you’re actively building your setup. We care about weekly action.

Do you take a cut? Is there a contract? ›

Founding Gen is built on collaboration + structure, not “agency control.” Any formal terms (if used) are meant to protect the group from ghosting/drama and set expectations.

If you’re shortlisted, we’ll explain the current policy clearly before you accept anything.

What about branding—can I function independently? What if I leave later? ›

You keep your identity and channels. Kuroveil branding is shared group branding. If you ever leave, you stop presenting as Kuroveil, but your personal brand remains yours.

Shortlisted applicants get the clear breakdown in the Discord chat.

How long does review take? ›

Rolling reviews. Expect 3–7 days. If shortlisted, you’ll be contacted via Discord.

What gets an application rejected fast? ›
  • Private/empty links, or no recent content
  • Vague answers / no goals
  • “Join and disappear” energy
  • Drama history or disrespectful behavior
